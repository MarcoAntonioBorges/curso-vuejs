#Projetos desenvolvidos no Curso de VueJS 2 da Udemy.

Usando diversas tecnologias, como:

- Vuetify
- Axios
- Bulma.io
- Vue-Router
- Material Design 
- Flux

Projeto compoem estruturas básicas de um projeto VueJS 2, como a ultilização de Componentes, ESLint, Vue Cli.

Para execução dos projetos, basta seguir os passos:

1. Entrar no diretorio raiz de alguns dos projetos.
2. Executar o npm -i, para instalação das dependencias, assim ele criara a pasta node_modules.
3. Executar o comando npm run dev, para a execução do projeto.
